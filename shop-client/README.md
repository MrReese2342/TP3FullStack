# Shop Client (React + Vite)

Frontend de l’application du TP3 – UE Full Stack.  
Cette application permet d’interagir avec l’API backend afin d’afficher,
rechercher et gérer les boutiques, via une interface web simple et responsive.

## Stack technique
- React
- Vite
- TypeScript
- HTML / CSS
- Fetch API

## Prérequis
- Node.js (version LTS recommandée)
- npm (ou un gestionnaire équivalent)

## Installation
Depuis le dossier `shop-client` :

```bash
npm install
npm run dev
